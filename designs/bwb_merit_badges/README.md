# BWB Merit Badge Tokens

Build the 24-token mixed badge sheet:

```bash
python3 scripts/laser_build.py --design bwb_merit_badges
```

The first revision uses 72 x 42 mm rounded tokens in a 4 x 6 grid. Nine badge types receive two or three copies through deterministic even allocation.

Typography uses Liberation Sans Bold titles and Regular descriptions. Physical readability and 0.18 mm hatch spacing remain calibration-only.
