# Queer Sex Party Merit Badge Tokens

Build the 24-token mixed badge sheet:

```bash
python3 scripts/laser_build.py --design queer_sex_party_merit_badges
```

The first revision uses 72 x 42 mm rounded tokens in a 4 x 6 grid. Six badge types receive four copies each.

Typography uses Liberation Sans Bold titles and Regular descriptions. Physical readability and 0.18 mm hatch spacing remain calibration-only.
